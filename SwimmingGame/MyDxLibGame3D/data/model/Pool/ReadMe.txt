■ステージ名：屋内プール（短水路）ver.0.8.2

　屋内プール（短水路）ステージです。屋内のみで外観はありません。可動ギミック
　搭載のためX形式アクセサリではなくPMXモデルとなっています。
　おまけとして監視台、ビート板整理棚、ベンチ、競技計測用時計等の備品が入って
　います。

　※MEMO※
　1) プール外周の排水溝テクスチャの表示が崩れる場合は、Furia様作成/配布のMME
　：「簡単適用〜ミップマップと異方向性フィルタ」を「排水溝」材質に個別適用す
　ることで緩和されるのを確認しています。

　2) 付属の水面アクセサリは間に合わせで作った残念なシロモノですので使用は「非
　」推奨です。プール本体に水面取付用ダミーボーンを搭載していますのでビームマン
　P様作成/配布の水面エフェクト（Water等）を直接取り付ける等してご利用ください。

　3) モーション作成時に邪魔にならないよう四方の壁と天井をモーフで消せるように
　してあります。ご活用ください。

　4) 室内灯はそぼろ様作成/配布のAutoLuminousやTCLXに対応しています。窓は透明度
　0の状態で若干発光するようにしてあります（窓の外の風景として広域ステージを配
　置する余裕がない場合を考慮）。

　5) ステージとしては大型ですのでシャドウ設定を行わないと全域にセルフシャドウ
　を落とすことができません。シャドウ設定必須です。GPUが強力であればExcellent
　Shadow等も。


★制限事項
　R18以上の動画での利用不可。
　利用者による改変は可。再配布は不可。
　商用利用不可。

　拙作モデル／アクセサリ等を使用していただいた動画／静画のマイリスト／マイクリ
　ップを作成するのをささやかな楽しみとしています。ご利用いただいた際はできるだ
　けコンテンツツリーへの配布動画の登録をお願いします。（通知が来るため探す手間
　なく視聴／閲覧できるためありがたいです）
